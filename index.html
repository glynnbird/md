If you would like your documents backed up to Cloudant, then simply add the URL of read/writable database in the form below.

  • Sign-up for a Cloudant account
  • Create a new database
  • In the permissions tab, generate a new api key
  • Make the generated api have "reader" & writer access
  • Enable CORS access
  • Construct the url: https://APIKEY:PASSWORD@HOSTNAME.cloudant.com/DATABASE_NAME
